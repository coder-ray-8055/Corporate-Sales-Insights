# Corporate-Sales-Insights
This repository contains the sales data for various electronic products for the year 2023. The dataset includes information about orders, products, categories, prices, quantities, regions, and payment modes.

## Dataset Details

- **File Name:** `sales_data.csv`
- **Format:** CSV (Comma-Separated Values)
- **Number of Rows:** 300+ orders
- **Columns:**
  - `Order_ID` : Unique identifier for each order
  - `Date` : Date of the order
  - `Product` : Name of the product sold
  - `Category` : Product category (Electronics / Accessories)
  - `Price` : Price of a single unit
  - `Quantity` : Number of units sold
  - `Region` : Region where the order was placed (North, South, East, West)
  - `Payment_Mode` : Payment method used (Card, UPI, COD)
  - `Day` : Day of the month
  - `Month` : Month of the year
  - `Year` : Year of the order
  - `Total_Sales` : Total sales amount for the order (Price Ã— Quantity)

## Purpose

This dataset can be used for:

- Analyzing sales trends by month, region, or category
- Generating sales reports
- Learning and practicing data analysis, visualization, and data science concepts
- Building predictive models for sales forecasting
